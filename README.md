# ClipMate marketing site

Static landing page for **ClipMate**, a Chrome extension for local clipboard history, search, and automatic categorization.

## Stack

- [Vite](https://vitejs.dev/) + React 18 + TypeScript
- [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/) components
- [React Router](https://reactrouter.com/) for client-side routing

## Scripts

| Command        | Description              |
| -------------- | ------------------------ |
| `npm run dev`  | Local dev server (Vite)  |
| `npm run build`| Production build         |
| `npm run preview` | Preview production build |
| `npm run lint` | ESLint                   |
| `npm test`     | Vitest (unit tests)      |

## Waitlist

The waitlist form demonstrates client-side validation and optional local persistence. Before launch, point submissions at your email provider or API and update the copy in `PrivacySection` and `Waitlist` accordingly.
